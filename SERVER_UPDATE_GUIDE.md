## 🔄 Код Шинэчлэх 5 Алхам

Сайтын кодонд өөрчлөлт оруулаад `git push` хийснийхээ дараа сервер дээрээ дараах 5 командыг дэс дарааллаар нь уншуулна.

### 1️⃣ Сервер рүү нэвтэрч, төслийн хавтас руу орох
Компьютерийнхээ Terminal / PowerShell дээрээс сервер рүү нэвтэрнэ:
```bash
ssh root@156.67.24.6
# Нууц үгээ оруулаад нэвтэрч орсны дараа:
cd /var/www/engiinshop
```

### 2️⃣ GitHub-аас шинэ кодоо татах
Төслийн хавтас дотроо байхдаа хамгийн сүүлийн өөрчлөлтүүдийг сервэрт татаж оруулна:
```bash
git pull
```

### 3️⃣ Сангууд болон Дата бааз шинэчлэх (Хэрэгцээтэй үед)
Хэрвээ та шинээр нэмэлт сан (npm package) суулгасан, эсвэл `prisma/schema.prisma` дотор дата баазын бүтцийн өөрчлөлт хийсэн бол заавал уншуулна:
```bash
npm install
npx prisma db push
```
*(Хэрэв ямар ч сан/бааз өөрчлөөгүй, зөвхөн дизайны өөрчлөлт хийсэн бол энэ алхмыг алгасаж болно).*

### 4️⃣ Төслийг шинээр Build хийх (Хамгийн чухал)
Татаж авсан шинэ кодоо интернэтэд ажиллахад бэлэн болгож хөрвүүлэх алхам. Хуучин гацсан кэш (cache) санах ойг цэвэрлэхийн тулд `.next` хавтсыг устгаад шинээр угсарна:
```bash
rm -rf .next
npm run build
```

### 5️⃣ Серверийг дахин ачаалж, өөрчлөлтийг идэвхжүүлэх
Бүх зүйл амжилттай угсрагдсаны дараа вэбсайтыг унтрааж, шинэ хувилбараар нь дахин асаана:
```bash
pm2 restart engiinshop
```

> ⚠️ **Анхааруулга:** Хэрэв та `.env` (орчны хувьсагч) дотор шинэ өөрчлөлт хийсэн бол 5-р алхам дээрх командыг `pm2 restart engiinshop --update-env` гэж заавал өөрчилж бичнэ үү!
