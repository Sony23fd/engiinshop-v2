# Хүргэлтийн Өдөр Сонгох Төлөвлөгөө

Хэрэглэгч сагс (/cart), шууд худалдан авалт (/product/[id]), болон захиалгын түүх (/track) хуудаснаас "Хүргэлтээр авах" сонголтыг хийхдээ админаас тохируулсан хүргэлтийн гарагуудаас (жишээ нь: Лхагва, Бямба) тодорхой өдрөө өөрөө сонгох боломжийг бий болгоно.

Сонгосон өдрийг системийн бүх хэсэгт (Админ хүргэлтийн жагсаалт гэх мэт) тохируулж, хүргэлтийн нэгдмэл, цэгцтэй байдлыг бий болгох бүрэн төлөвлөгөөг доор танилцуулж байна.

## Proposed Changes

### 1. Delivery Date Logic (Огноо тооцоолох)

Админы тохиргооноос (жишээ нь "3,6" буюу Лхагва, Бямба) дараагийн 2-3 боломжит хүргэлт гарах өдрийг тооцоолж гаргадаг функц бичнэ.
#### [MODIFIED] `src/lib/utils.ts`
- `getUpcomingDeliveryDates(scheduleDaysStr: string, count: number = 2): Date[]` функц нэмнэ.

---

### 2. Storefront (Хэрэглэгчийн тал: Сонголт хийх)

Хэрэглэгч "Хүргэлтээр авах" сонголтыг идэвхжүүлмэгц, доор нь гарах боломжтой ойрын хүргэлтийн өдрүүд "Radio Button" хэлбэрээр харагдах бөгөөд хэрэглэгч нэгийг нь заавал сонгодог байна.

#### [MODIFIED] `src/app/(storefront)/cart/CartClient.tsx`
- `wantsDelivery` идэвхжсэн үед `selectedDeliveryDate` state ажиллана.
- Хэрэглэгчид "Хүргүүлэх өдөр сонгох" хэсэг гарч ирнэ.
- Form submit хийгдэхэд сонгогдсон өдрийг `createOrder` рүү `deliveryDate` байдлаар дамжуулна.

#### [MODIFIED] `src/app/(storefront)/product/[id]/ProductOrderForm.tsx`
- Худалдан авалтын form дээр Cart-тай яг ижил "Хүргүүлэх өдөр сонгох" Radio group нэмнээ.
- Худалдан авалтын үед баталгаажуулалтанд оруулна.

#### [MODIFIED] `src/app/(storefront)/track/DeliveryRequestButton.tsx`
- Хэрэглэгч бараа нь Улаанбаатарт ирсний дараа хүргэлт захиалах үед гарч ирдэг Modal дотор "Хүргүүлэх өдөр сонгох" хэсгийг нэмнэ.
- `requestDelivery` action-д сонгосон өдрөө дамжуулах.

---

### 3. Server Actions (Бааз руу хадгалах)

Явуулсан огноог баазад `deliveryDate` (Prisma-ийн `DateTime` төрөл) талбарт хадгалж авах логикийг нэмнэ.

#### [MODIFIED] `src/app/actions/storefront-actions.ts` эсвэл `order-actions.ts` (API функц)
- `createOrder` функцэд `deliveryDate` параметр хүлээж авах өөрчлөлт.
- `requestDelivery` функцэд `deliveryDate` параметр хүлээж авах өөрчлөлт.
- Prisma `db.order.create/update` хийх үед тус огноог бичиж хадгална.

---

### 4. Admin Dashboard (Админд харагдах байдал)

Админ талд хүргүүлэхээр өгсөн захиалгуудын жагсаалтанд хэрэглэгчийн сонгосон огноог харуулж, тэр өдрөөр нь хүргэлтээ бэлдэх боломжийг олгоно.

#### [MODIFIED] `src/app/admin/orders/delivery/page.tsx`
- "Захиалагчийн хүссэн огноо: 2026.04.08, Лхагва" гэх зэргээр тод харуулна.

---
