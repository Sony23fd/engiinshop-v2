# Ubuntu VPS Сервер дээр автомат цуцлалт тохируулах заавар

Энэхүү заавар нь 24 цаг хэтэрсэн захиалгуудыг автоматаар цуцлах процессыг таны Contabo сервер дээр идэвхжүүлэхэд тусална. 

## 1. Үйлдлийн системийг шалгах (Сонголтоор)
Терминал дээрээ дараах командыг ажиллуулж Ubuntu мөн эсэхийг баталгаажуулна уу:
```bash
cat /etc/os-release
```

## 2. Нууц түлхүүр тохируулах
Аюулгүй байдлын үүднээс бид `CRON_SECRET` ашиглана. 

1. Сервер дээрээ өөрийн төслийн хавтас руу орно.
2. `.env` файлыг нээнэ (жишээ нь `nano .env` командаар).
3. Дараах мөрийг нэмнэ:
   ```env
   CRON_SECRET=tany_nuuts_tulkhuur_123
   ```
   *(Жич: `tany_nuuts_tulkhuur_123`-ийг өөрийн хүссэн дурын нууц үгээр сольж болно)*

## 3. Системийг дахин ачаалах
Шинэ тохиргоог идэвхжүүлэхийн тулд аппликейшнийг дахин ачаалах шаардлагатай. Хэрэв та `PM2` ашигладаг бол:
```bash
pm2 restart all
```

## 4. Crontab тохируулах (Автоматжуулалт)
Энэ алхам нь бидний үүсгэсэн API-г цаг тутамд автоматаар дуудаж байх тохиргоо юм.

1. Терминал дээр дараах командыг бичнэ:
   ```bash
   crontab -e
   ```
   *(Хэрэв анх удаа нээж байгаа бол редактор сонгохыг хүснэ. `1` буюу `nano`-г сонгоорой)*

2. Файлын хамгийн доор дараах мөрийг нэмнэ:
   ```bash
   0 * * * * curl -H "Authorization: Bearer tany_nuuts_tulkhuur_123" http://localhost:3000/api/cron/cancel-expired-orders
   ```
   *(Жич: `tany_nuuts_tulkhuur_123`-ийг өөрийн өмнөх алхамд тохируулсан нууц үгээр сольсон эсэхээ шалгаарай)*

3. Хадгалаад гарна (Ctrl+O, Enter, Ctrl+X).

---

> [!TIP]
> **Турших:** Тохиргоо зөв болсон эсэхийг терминал дээрээ саяны `curl ...` командыг шууд хуулж тавиад ажиллуулж үзээрэй. Хэрэв `{"success":true,...}` гэсэн хариу ирж байвал бүх зүйл зөв байна гэсэн үг.
